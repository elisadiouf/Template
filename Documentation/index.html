Layout position

You can Chnage the initial layout by setting control classes to the <html> element. 

index.html
<html class="layout-fixed layout-navbar-fixed">
Class Name Description
.layout-fixed Set layout Side menu position to fixed.
.layout-navbar-fixed Set layout Header position to fixed.

Layout position

You can Chnage the initial layout by setting control classes to the layout-navbar id. 

index.html
<div id="layout-sidenav" class="logo-dark">
Class Name Description
.logo-dark Set Brand background to dark color.
.logo-white Set Brand background to light color.
